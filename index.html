About This Project

Vibe Games is a private, educational sandbox built to explore modern web development workflows. The core goals include:

  • Vibe Coding & AI: Experimenting with AI-assisted software development to understand its practical processes and limitations.
  • Hosting Preparation: Gaining hands-on experience with static deployments, server management, and web infrastructure.
  • Just for Fun: Building and playing classic, lightweight browser mini-games.

This project is non-commercial. All code is open-source and released under the MIT License.
